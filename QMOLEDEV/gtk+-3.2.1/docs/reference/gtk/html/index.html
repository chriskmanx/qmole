GTK+ 3 Reference Manual

This document is for the GTK+ 3 library, version 3.2.1 . The latest versions can be found online at http://library.gnome.org/devel/gtk3/. If you are looking for the older GTK+ 2 series of libraries, see http://library.gnome.org/devel/gtk/.
