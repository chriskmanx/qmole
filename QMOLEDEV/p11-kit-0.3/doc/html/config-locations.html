Configuration Files

Each configured PKCS#11 module is has its own config file. These files are placed in a directory. In addition a global config file exists. There is a system configuration consisting of the various module config files and a file for global configuration. Optionally each user can provide additional configuration or override the system configuration.

The system global configuration file is in /etc/pkcs11/pkcs11.conf and the user global configuration file is in ~/.pkcs11/pkcs11.conf in the user's home directory.

The module config files are located in the /etc/pkcs11/modules directory, with one configuration file per module. In addition the ~/.pkcs11/modules directory can be used for modules installed by the user.
